# AWS_Data_Analysis

AWS를 이용한 부동산 데이터 분석 입니다

## 실행 환경

   - 클라우드 서버
      - AWS EC2
      - Ubuntu Linux 22.04 LTS
      - Anaconda 2023.07
   - 파일 서버
      - AWS S3
   - 화면 대시보드
      - Python Streamlit
